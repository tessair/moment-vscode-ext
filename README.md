## Build and Package

To build and package the extension using the 'vsce' tool, follow these steps:

1. Install dependencies:
   ```sh
   npm install
   ```

2. Compile the extension:
   ```sh
   npm run compile
   ```

3. Package the extension:
   ```sh
   vsce package
   ```

4. Publish the extension (requires authentication):
   ```sh
   vsce publish
   ```
